# ⚡ DISCORD TOKEN GENERATOR v1 (unmaintained)

## 🚀 Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/anomusly/Discord-Token-Retriever.git
   cd Discord-Token-Retriever
   ```
2. **Install Requirements:**

   ```bash
   pip install -r requirements.txt
   ```
3. **Run the tool:**

   ```bash
   python main.py
   ```

## 🎯 About

A simple and efficient **Discord token generator** that extracts tokens from email:password combinations automatically.
It supports bulk processing, handles rate limits, and saves organized output files.

> ⚠️ This project is for **educational and testing purposes only**.
> Misuse of this tool can result in account termination.
